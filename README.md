# DevOps-Portfolio

Willkommen zu meinem DevOps Portfolio! 🚀  

Dieses Repository zeigt meine Projekte und Fähigkeiten in den Kernbereichen von DevOps:

- **Cloud** – Deployment von Anwendungen und Infrastruktur auf AWS  
- **Docker & Kubernetes** – Containerisierung und Orchestrierung von Anwendungen  
- **CI/CD** – Automatisierte Build- und Deployment-Pipelines  
- **Terraform** – Infrastructure as Code für reproduzierbare Deployments  

Alle Projekte enthalten Code, Konfigurationen, Screenshots und kurze Beschreibungen, um den Aufbau, die Funktionsweise und die eingesetzten Technologien zu erklären.

---

 **Ziel:**  
Dieses Portfolio zeigt praxisnahe DevOps-Fähigkeiten und dient als Referenz für Bewerbungen im Bereich Junior DevOps Engineer.
